Vous devez installer NodeJS.

Lorsque ceci est fait vous devez ouvrir un shell (La touche Windows + R et ensuite tapez cmd)
Une fois dans l'invite de commande de windows. vous devez tapez ce qui suit :

npm install mysql

Ceci installe le module pour communiquer avec une base de données MySQL.

Dans le dossier node :
mysqlTest.js : Montre comment se connecter avec une base de données
server.js : Montre comment créer une serveur HTTP
TestConnection : Utilise les deux précédents exercice pour faire une connection MySQL
 en utilisant un serveur HTTP pour retourner le résultat.

Le HTTPConnection est un projet android qui se connecte au serveur http (créer avec NodeJS)
le projet utilise un Thread (ce que nous avons pas vu en android encore) et HttpUrlConnection
